## Overview
This project demonstrates a basic AI model deployment workflow using a pre-trained
Transformer model. The application downloads a model, loads it using Hugging Face
Transformers, and performs inference using PyTorch.

## Model Used
- **DistilBERT** (fine-tuned on SST-2 dataset)
- **Task:** Sentiment Analysis (Positive / Negative)

## Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
- Requests
